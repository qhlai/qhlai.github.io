周一

入职，拿回车子。装新的vmware。

找镜像先把环境搭好。

vmware install

vmware下载地址：

https://www.vmware.com/cn/products/workstation-pro/workstation-pro-evaluation.html

vmware激活许可证：ZF3R0-FHED2-M80TY-8QYGC-NPKYF

ubuntu init

ubuntu 有很多种桌面版，对于系统我自己喜欢用的是kde，建议新手用gnome，可能网上中文资料多点。

ubuntu (gnome)

https://releases.ubuntu.com/18.04.6/ubuntu-18.04.6-desktop-amd64.iso

kubuntu (kde)

https://cdimage.ubuntu.com/kubuntu/releases/18.04/release/kubuntu-18.04.5-desktop-amd64.iso

换源

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
sudo sed -i 's/archive.ubuntu.com/mirrors.ustc.edu.cn/g' /etc/apt/sources.list
#nano /etc/apt/sources.list
sudo apt update && sudo apt upgrade
# 常用软件
sudo apt install -y net-tools curl wget htop powertop
sudo apt autoremove

ros install

sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ $DISTRIB_CODENAME main" > /etc/apt/sources.list.d/ros-latest.list'
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
sudo apt update
sudo apt install -y ros-melodic-desktop-full
sudo apt-get install ros-melodic-desktop-full
sudo apt-get install -y ros-melodic-rqt*

sudo apt install -y python-rosdep2
sudo rosdep fix-permissions
rosdep update #github 被干扰严重多试几次 或者参考
#https://blog.csdn.net/QLeelq/article/details/119041164
#https://mp.weixin.qq.com/s/VGs8oWdhHH6XsHcx21lN4Q
#或者直接上软路由
sudo rosdep init
